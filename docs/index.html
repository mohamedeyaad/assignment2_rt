scripts.action_client.odom_callback(msg)[source]

Callback to update robot position and velocity.

This function is triggered whenever a new Odometry message is received. It updates the global variables robot_position and robot_velocity and publishes the robot’s state.

Parameters:

msg (nav_msgs.msg.Odometry) – The Odometry message containing the robot’s position and velocity.

scripts.action_client.send_goal(client, x, y)[source]

Send a goal to the action server.

This function creates a PlanningGoal message with the specified target position and sends it to the action server.

Parameters:

  • client (actionlib.SimpleActionClient) – The action client used to send the goal.

  • x (float) – The x-coordinate of the goal position.

  • y (float) – The y-coordinate of the goal position.
